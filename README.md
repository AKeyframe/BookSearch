# Bookworm's Bible
---
An online app that allows users to search for books and add them to their reading list. 

---
## Technologies Used
---
 - HTML5
 - CSS3
 - JavaScript
 - jQuery
 - Google Books API - Can be found [here](https://developers.google.com/books/docs/overview).
 - Google Fonts
